Multiple Select

Multiple select is a jQuery plugin to select multiple elements.

Download »

Examples

The Default

Example Code

<head>
    <link href="multiple-select.css" rel="stylesheet"/>
    <script src="jquery.multiple.select.js"></script>
    <script>
        $(function() { 
            $("#select1").multipleSelect(); 
        });
    </script>
</head>
<body>
    <select id="select1">
        <option value="1">January</option>
        ...
        <option value="12">December</option>
    </select>
</body>

The Multiple Items

Example Code

<head>
    <link href="multiple-select.css" rel="stylesheet"/>

    <select id="select3">

Programmatic Access

Multiple Select supports methods that allow programmatic control of the component

Example Code

<head>
    <link href="multiple-select.css" rel="stylesheet"/>
    <script src="jquery.multiple.select.js"></script>
    <script>
        $(function() { 
            $("#select2").multipleSelect({
                isopen: true,
                selectAll: false
            });
            $('#setSelects').click(function() {
                $('#select2').multipleSelect('setSelects', [1, 3]);
            });
            $('#getSelects').click(function() {
                alert($('#select2').multipleSelect('getSelects'));
                alert('Selected text is: ' + $('#select2').multipleSelect('getSelects', 'text'));
            });
            $('#enable').click(function() {
               $('#select2').multipleSelect('enable');
            });
            $('#disable').click(function() {
               $('#select2').multipleSelect('disable');
            });
        });
    </script>
</head>
<body>
    <select id="select2">
        <option value="1">First</option>
        ...
        <option value="4">Fourth</option>
    </select>
</body>

Documentation

Constructor

Parameter Type Description
isopen boolean

Whether or not Multiple Select open the select dropdown.

By default this option is set to false.
selectAll boolean

Whether or not Multiple Select show select all checkbox.

By default this option is set to true.
selectAllText string

Multiple Select select all checkbox text.

By default this option is set to Select all.
multiple boolean

Whether or not Multiple Select show multiple items in a row.

By default this option is set to false.
multipleWidth integer

Multiple Select show multiple items width.

By default this option is set to 80.

getSelects

Gets the selected values.

Parameter Type Description
type string

The type of selected items. value or text

The default is value.

Example: 

alert('Selected value is: ' + $('#select2').multipleSelect('getSelects'));
alert('Selected text is: ' + $('#select2').multipleSelect('getSelects', 'text'));

setSelects

Sets the selected values.

Parameter Type Description
values array

The values of selected items.

Example: 

$('#select2').multipleSelect('setSelects', [1, 3]);

disable

Disables Select. During this mode the user is not allowed to manipulate the selection.

enable

Enables Select.
